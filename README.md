# SillyTavern 用户人设分组 (Persona Groups)

为 SillyTavern 的"用户人设"功能增加分组管理 + 快捷切换能力。
让数十甚至上百个 Persona 易于管理。

## 功能

### 位置一：用户设定面板（顶栏「用户设定」）
- 📁 **分组管理**：新建、重命名、删除分组；分组可折叠/展开
- ☑ **多选模式**：批量将人设移入分组或移出
- 🔍 **筛选**：全部/已绑定/未绑定/按分组
- 📄 **自定义分页**：5/10/25/50/100/200，含首页/末页快速跳转
- 👁️ **隐藏分组视图**：一键隐藏全部分组，简化视图

### 位置二：底栏快捷弹窗
- 输入栏旁的小头像，点击弹出快速切换面板
- 与位置一的分组结构同步
- 移动端友好，弹窗位置自适应

## 安装

通过 ST 的"安装扩展"功能输入仓库地址

## 设置

在扩展面板找到"**用户人设分组**"，可以开关位置二的快捷弹窗。
